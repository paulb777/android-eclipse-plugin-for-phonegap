JavaScript Development Toolkit (JSDT)

Embeded JavaScript

  To edit JavaScript embeded in HTML or JSP create a project that supports JSP or HTML. Below are instructions for a Static Web Project (HTML + JavaScript) but many other project types will work.

JSDT Requires no additional configuration for WST based projects that support HTML or JSP. (Unless the project was created before the JSDT was installed. More info**)

1. Select File --> New--> Project
 

2. Expand the Web folder and choose Static Web Project from the New Project Wizzard Dialog.
 

3. Type a project name in the Project name field and click Finish.
 


** Any project created before the JSDT plugins are installed must be re-created in order to support the JSDT functionality
